Au Jour Le Jour

Offline-first SQLite Active
Contact Projects

Au Jour Le Jour is a local-first essentials tracker that runs fully offline with a local SQLite file. It provides daily backups, CSV/JSON exports, and a v1 integration contract for external tools.

What It Solves

  • Tracks essentials month by month without cloud dependency.
  • Creates daily SQLite backups and exportable CSV/JSON.
  • Integrations use the v1 contract instead of direct DB access.

Status

Active local-first build with a v1 integration contract.
