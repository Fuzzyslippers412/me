Project

Au Jour Le Jour

Offline-first SQLite Active
Visit Au Jour Le Jour Contact Projects

Au Jour Le Jour is a local-first essentials tracker with an offline SQLite ledger for the local server and a browser-only PWA that stores data in IndexedDB. Month generation is idempotent, partial payments and cash-on-hand are tracked per month, and it provides daily backups plus CSV/JSON export with a v1 integration contract for MyCasaPro’s Finance Agent (Mamdou).

What It Solves

  • Local server with SQLite and a browser PWA (IndexedDB) keep data on-device.
  • Idempotent month generation with partial payments and cash-on-hand.
  • Daily backups plus CSV/JSON export for portability.
  • v1 integration contract for agent summaries and actions.

Status

Live at aujourlejour.xyz, with the local server option for the full offline ledger.
