Project

ChattyPatty

Decentralized compute LLM inference In build
Visit ChattyPatty Contact

Decentralized compute marketplace for LLM inference with a Sonic-first trust plane, deterministic receipts, exclusive leasing by default, and hardened runtime isolation. An OpenAI-compatible gateway routes workloads to staked providers through escrow and verifiable receipts.

What It Solves

  • Verifiable receipts, challenges, and slashing enforce quality.
  • Exclusive leasing by default for predictable capacity and routing.
  • Hardened runtime isolation for secure provider nodes.
  • OpenAI-compatible gateway with token accounting and rate controls.

Status

Specs locked. Contract interfaces scaffolded. Canonicalization rules, protocol drafts, and core schemas in place. Go services scaffolded with request validation and receipt submission.

Independent software builder working on local-first and privacy-first systems.
