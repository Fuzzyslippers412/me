ChattyPatty

Calcolo decentralizzato Inferenza LLM In costruzione
Visita ChattyPatty Contatto

Marketplace di calcolo decentralizzato per inferenza LLM con trust plane Sonic-first, ricevute deterministiche, leasing esclusivo di default e isolamento runtime rinforzato. Un gateway compatibile con OpenAI instrada i job verso provider in stake tramite escrow e ricevute verificabili.

Cosa risolve

  • Ricevute verificabili, challenge e slashing garantiscono la qualità.
  • Leasing esclusivo di default per capacità prevedibile e routing.
  • Isolamento runtime rinforzato per nodi provider sicuri.
  • Gateway compatibile con OpenAI con accounting dei token e controlli di rate.

Stato

Specifiche bloccate. Interfacce dei contratti scaffoldate. Regole di canonicalizzazione, bozze di protocollo e schemi core pronti. Servizi Go scaffoldati con validazione richieste e invio delle ricevute.

Costruttore software indipendente orientato a sistemi local-first e privacy-first.
