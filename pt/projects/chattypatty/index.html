Projeto

ChattyPatty

Computação descentralizada Inferência LLM Em laboratório
Visitar ChattyPatty Contato

Marketplace de computação descentralizada para inferência LLM com trust plane Sonic-first, recibos determinísticos, leasing exclusivo por padrão e isolamento de runtime reforçado. Um gateway compatível com OpenAI roteia workloads para provedores em staking via escrow e recibos verificáveis.

O que resolve

  • Recibos verificáveis, challenges e slashing garantem qualidade.
  • Leasing exclusivo por padrão para capacidade previsível.
  • Isolamento de runtime reforçado para nós provedores seguros.
  • Gateway compatível com OpenAI com contabilidade de tokens e controles de taxa.

Status

Especificações fechadas. Interfaces de contratos esboçadas. Regras de canonicalização, rascunhos de protocolo e esquemas core prontos. Serviços Go scaffoldados com validação de requisições e envio de recibos.

Construtor de software independente, sistemas local-first e focados em privacidade.
