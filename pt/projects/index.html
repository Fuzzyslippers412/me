MyCasaPro

MyCasaPro é uma plataforma local-first de coordenação e verificação para proprietários e prestadores. Ela acompanha marcos, evidências, aprovações e pagamentos com um registro de eventos inviolável e ligação opcional com pagamentos privados.

Theo.farm

Theo.farm é uma ferramenta local-first para pequenas fazendas e produtores gerenciarem horários, tarefas, estoque e notas de campo offline-first com sincronização quando disponível.

GhostProtocol

GhostProtocol é uma UI amigável para agentes para enviar transações privadas on-chain. Conecte uma carteira, escolha uma rede e, se o Railgun for suportado, ele guia o shielding, a transferência privada, o unshielding e cria um registro verificável de ligação de pagamento.

ChattyPatty

ChattyPatty é um marketplace distribuído de computação de IA onde as pessoas compartilham a potência ociosa de laptops e desktops para executar modelos de IA para outros. Ele roteia cargas de trabalho, mede o uso e torna o acesso à IA mais barato para usuários sem hardware de ponta.

Soundcheck.AI

Soundcheck.AI usa IA para analisar registros financeiros públicos de ONGs e escolas (incluindo registros do IRS) para revelar padrões de gastos e verificar se as organizações gastam o dinheiro como afirmam.
